# OPENAZV Ecosystem

OPENAZV is a modular multi-agent ecosystem scaffold with:
- API orchestrator (FastAPI)
- Parallel task execution
- NeuroCausal retrieval pipeline skeleton
- Startup scripts for local development

## Quick start

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r openazv/requirements.txt
python openazv/start_openazv.py
```

Then open: http://127.0.0.1:8088/docs
