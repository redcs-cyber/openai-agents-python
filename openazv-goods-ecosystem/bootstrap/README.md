# Bootstrap

Bu klasör, içerikte geçen teknoloji ve servislerin başlangıç envanterini ve hızlı kurulum yolunu sağlar.

## Dosyalar

- `stack-catalog.yaml`: Projede adı geçen AI/servis/kütüphane kategorik envanteri.
- `requirements-starter.txt`: Yerel PoC başlatmak için başlangıç Python bağımlılıkları.
- `start.sh`: Sanal ortam kurup bağımlılıkları yükleyen script.

## Başlatma

```bash
cd openazv-goods-ecosystem
./bootstrap/start.sh
```

> Not: Harici servisler (OpenAI, Twilio, Google, vb.) için API anahtarları ve hesap yapılandırmaları ayrıca gereklidir.
